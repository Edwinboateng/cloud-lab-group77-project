About This Project

This website was built as part of our Cloud Computing Practical Lab 1 assignment at GCTU. The goal was to work collaboratively as a team to design, build, and deploy a static website using HTML, CSS, and JavaScript, then host it live using GitHub Pages.


Through this project, we practiced real-world skills including version control with Git, team collaboration via GitHub repositories, and deploying websites to the cloud — all core skills in modern cloud computing.
