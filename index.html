About This Project

☁️

This website was built as part of our Cloud Computing Practical Lab 1 assignment at GCTU. Working together as a team, we designed, built, and deployed a static website using HTML, CSS, and JavaScript, then hosted it live on AWS S3.


Through this project, we gained real hands-on experience with version control using Git, team collaboration through GitHub repositories, and deploying websites to the cloud. These are core skills every modern cloud computing professional needs.
