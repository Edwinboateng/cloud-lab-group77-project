Project Description

This project is a static website developed for Cloud Computing Practical Lab 1. Our team worked together to design and build a multi-page website from scratch using core web technologies — HTML for structure, CSS for styling, and JavaScript for interactivity.


The website was then deployed using GitHub Pages, giving us hands-on experience with cloud-based deployment and collaborative version control using Git and GitHub.
