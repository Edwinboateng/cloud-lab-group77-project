Project Description

This project is a static website developed for Cloud Computing Practical Lab 1 at GCTU. Our team worked together to design and build a multi-page website from scratch using core web technologies. HTML gave us the structure, CSS handled the styling, and JavaScript added interactivity.


The website was then deployed on AWS S3, giving us real hands-on experience with cloud-based hosting, public access configuration, and bucket policies. We also practiced collaborative version control using Git and GitHub throughout the entire process.
